## [2.0.0] - 2026-04-28

### Added
- **Memorial Profiles**: Create and manage digital memorial profiles for loved ones
- **AI Chat (Mimo)**: Conversational AI companion powered by Mimo v2 for remembrance and reflection
- **Voice Calls (Twilio + Mimo TTS)**: Receive phone calls with AI-generated voice conversations
- **Video Generation (Mimo Omni)**: Generate memorial tribute videos with AI narration
- **Stripe Billing**: Four subscription tiers (Seed, Tree, Garden, Family) with Stripe Checkout
- **Web Application**: Full-featured web UI served on port 8102
- **SQLite Database**: Local storage for memorial data, chat history, and user profiles
- **FFmpeg Integration**: Video processing pipeline for tribute video creation
- **CORS Configuration**: Configurable allowed origins for production deployment
- **Environment Configuration**: `.env.example` with all required and optional settings
- **Security Policy**: Vulnerability reporting and operator best practices
- **Contributing Guide**: Developer setup, code style, and PR workflow documentation
