# 念念 Eterna 品牌短片分镜

## 模式

动画模式，情绪克制，写实 UI 叙事，不做夸张卡通，不做廉价催泪。

## 核心创意

- 视觉主张：深夜的静、记忆的暖、手机像一道被轻轻打开的入口。
- 内容主张：不是“复活谁”，而是把关系中的回应感重新带回来。
- 动效主张：慢镜头漂移、柔和发光、节拍点上的 UI 进入、字幕和旁白同步推进。

## 片长

24 秒，16:9，动画品牌短片。

## 受众感受目标

先感到安静的失落，再感到被接住，最后理解产品不是噱头，而是一种长期陪伴服务。

## 画面风格

- 夜色蓝黑为主基调，暖杏色做情感高光。
- 镜头始终克制，移动幅度小，像呼吸一样。
- UI 不是主角，但所有情绪都通过 UI 被具体化。
- 屏幕文字不堆砌，每个镜头只承担一个信息任务。

## 旁白表演指导

- 声线：30 至 40 岁中文女声，克制、温柔、低气压但不虚弱。
- 情绪：不是哭腔，也不是广告播音腔，更像夜里轻声讲一个真事。
- 节奏：前半慢，后半略微抬起希望感，但始终不外放。
- 停顿：每句前半给观众留半拍理解空间，重要词如“回应”“入口”“那一刻”“延续”要轻微压重音。

## 音乐方向

- 类型：钢琴 + 柔和 pad + 极轻弦乐纹理。
- BPM：80。
- 拍号：4/4。
- 结构：8 小节，每小节 3 秒，总长 24 秒。
- 目标：像心跳一样稳定，不煽情抢戏，永远服务于旁白和画面。

## 音乐层次建议

- 0s - 6s：单音钢琴 + 很薄的环境氛围。
- 6s - 12s：加入轻颗粒钟琴，表现“入口被打开”。
- 12s - 18s：弦乐 pad 提升半级情绪，回复出现时补一层暖低频。
- 18s - 21s：节拍更清晰，但音量不明显增加。
- 21s - 24s：留下品牌主音，最后半秒自然收尾，不要硬切。

## 镜头与动画执行原则

- 所有 UI 出现都要贴拍，但不要机械弹跳。
- 文字尽量走淡入和滑入，不做夸张缩放。
- 回复气泡要比用户气泡更稳定、更柔和，体现“被接住”的感觉。
- 服务卡片不需要解释所有功能，只要证明“纪念能继续发生”。
- 结尾 CTA 不说价格，不说技术，只说关系。

## 可直接交付给动效师的重点

- 保留当前首页视频的色彩系统与产品语言。
- 每个镜头只讲一件事，不追加说明型 UI。
- 所有拍点以音乐强拍为准，字幕和旁白可以比拍点早 2 到 4 帧进入。
- 如果后续加入真人配音，优先根据旁白停顿微调镜头，而不是反过来压旁白。
